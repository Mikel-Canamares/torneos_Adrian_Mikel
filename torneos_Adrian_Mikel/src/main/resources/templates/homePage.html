Proyecto Torneos

Intrucciones creacion y borrado de datos

  • Para crear las tablas debes mirar antes si se necesita algun dato de otra tabla, por ejemplo, para crear un departamento necesitas crear previamente una ciudad y una federacion. Al escoger cualquiera de estos ultimos valores tienes un desplegable
  • Si intentas crear cualquier tupla sin introducir algun campo obligatorio te saltará un mensaje en rojo diciendote que debes rellenar ese campo
  • En los campos de teléfono solo te permite introducir numeros, si introduces cualquier otro caracter te saltara un mensaje de error en rojo
  • Para añadir equipos a un torneo debes primero crear un torneo, despues darle al boton de "añadir equipos", en la pantalla que se muestra podrás ir seleccionando los equipos disponibles creados para agregarlos
  • Puedes borrar cualquier fila, si borras por ejemplo una ciudad, que es una tabla cuya clave se propaga en muchas otras, las tuplas de las tablas donde se haya propagado la clave de ciudad se borrarán
